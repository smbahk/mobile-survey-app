범주	기능 설명
위치 수집	3·5·10초 동안 GPS 수신, 가장 정확한 위치 저장
GeoJSON 생성	조사자 이름 + 장소명 + 정확도 + 도로명주소 포함
사진 첨부	원경 / 근경 촬영 및 저장
지도 시각화	오늘 수집점(빨강) + 기존 점(색상별) + 텍스트 표기
클러스터링	마커가 많을 경우 자동 묶기
카카오맵 기능	확대/축소, 스카이뷰/지도 토글, 지적편집도 레이어
GeoJSON 분석 리포트	조사자별 건수, 평균 정확도, 장소 목록
PDF 출력	요약 정보 및 장소 목록 자동 PDF 저장
차트 시각화	조사자별 건수 막대그래프 (Chart.js)
모바일 최적화	전체 UI 반응형 + 직관적 구성
